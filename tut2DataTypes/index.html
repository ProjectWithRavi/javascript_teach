Data Types in JavaScript

There are two types of data types in JavaScript.

  1. Primitive

  2. Non Primitive

primitive data types.

These are predefined data types and there is no need to create them in JavaScript

primitive data type एक सिंगल वैल्यू को represent करता है जो एक फिक्स वैल्यू के साथ होती है और बिल्ट इन का मतलब होता है की यह पहले से ही डिफाइन होती है । जिसमे बेसिक डाटा टाइप string, number , Boolean , null , undefined, bigint , symbo आते हैं

We can classify primitive data types in 7 categories.

  1. string --> string is a sequence of characters and can contain letters, numbers, symbols and even spaces. It must be enclosed in quotation marks for it to be recognized as a string.
  2. number
  3. boolean
  4. undefined --> A variable that has not been assigned a value is of type undefined
  5. null --> In JavaScript, null and undefined represent the absence of a value. null is an assignment value that represents no value or no object
  6. bigint (home work)
  7. symbol (home work)

Non primitive data types.
