Dialog Boxes

In JavaScript, prompt(), confirm(), and alert() are used to interact with the user by displaying dialog boxes, but they serve different purposes:

There are three types of Dialog Boxes

  1. alert("Your message goes here.")
    alert(): Used to display a simple message to the user. It shows an alert box with a specified message and an "OK" button. The user cannot interact with the page until they click "OK."

  2. confirm("Your message goes here.")
    Used to ask the user for confirmation. It shows a dialog box with a message and "OK" and "Cancel" buttons. It returns true if the user clicks "OK" and false if they click "Cancel."

  3. prompt("Your message goes here.")
    Used to ask the user for input. It displays a dialog box with a message, an input field, and "OK" and "Cancel" buttons. If the user clicks "OK," it returns the input value as a string. If they click "Cancel," it returns null.
