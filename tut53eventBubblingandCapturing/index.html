Event Bubbling and Capturing

event bubbling javascript का एक मैकेनिज्म है जिसमें एक इवेंट अपने टारगेटेड एलिमेंट से start होता है और डोम hierarchy में अपने पैरेंट तक जाता हैं (targetedElement--> body --> document --> window )

event capture javascript propagation process का एक फेस है जिसमें इवेंट सबसे पहले रूट element पर ट्रिगर होता है और अपने टारगेटेड एलिमेंट तक जाता हैं (window --> document --> body --> targetedElement )
